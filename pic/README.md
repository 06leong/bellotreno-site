# 服务类型图片说明

本文件夹需要包含以下PNG格式图片文件，用于显示各种列车服务类型标识：

## Trenitalia 列车服务类型

- **IC.png** - Trenitalia Intercity（城际列车）
- **ICN.png** - Trenitalia Intercity Notte（夜间城际列车）
- **RV.png** - Trenitalia Regionale, Regionale Veloce, Metropolitan（区域列车、快速区域列车、都市列车）
- **FR.png** - Trenitalia Frecciarossa（红箭高速列车）
- **FA.png** - Trenitalia Frecciargento（银箭高速列车）
- **FB.png** - Trenitalia Frecciabianca（白箭高速列车）
- **EC.png** - Trenitalia Eurocity（欧洲城际列车）
- **EN.png** - Trenitalia EuroNight（欧洲夜车）

## 区域运营商列车服务类型

- **regn.png** - 用于以下运营商的区域列车：
  - Trenitalia TPER (Regionale, Regionale Veloce)
  - Ferrovie del Sud Est (Regionale, Regionale Veloce)
  - Trenord (Regionale)

## ÖBB 列车服务类型

- **RJ.png** - ÖBB Eurocity（欧洲城际列车）
- **NJ.png** - ÖBB EuroNight（欧洲夜车）

## 图片要求

- 格式：PNG（支持透明背景）
- 建议尺寸：高度约 80-120px（宽度自适应）
- 样式：官方Logo或标识
- 背景：透明或白色

## 获取官方Logo

您可以从以下官方网站获取Logo：
- Trenitalia: https://www.trenitalia.com
- Trenitalia TPER: https://www.trenitaliatper.it
- ÖBB: https://www.oebb.at
- Trenord: https://www.trenord.it
